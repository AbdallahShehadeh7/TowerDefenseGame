# Tower Defense Game

## Project Description

This is a Java tower defense game made for my final project. The player places towers on the grass to stop enemies from reaching the end of the path. The towers automatically shoot enemies when they come close enough.

## How the Game Works

- Enemies move along the brown path.
- The player clicks on green grass tiles to place towers.
- Towers shoot enemies automatically.
- Defeating enemies gives the player money.
- If enemies reach the end of the path, the player loses lives.
- The game ends when the player has no lives left.
- Each wave becomes harder.

## Controls

- Click a green tile to place a tower.
- Click `Start Wave` to begin the next wave.
- Click `Reset` to restart the game.

## Features

- Java Swing window
- Enemy waves
- Tower placement
- Automatic tower shooting
- Money system
- Lives system
- Game over screen
- Reset button

## How to Run the Project

1. Open IntelliJ IDEA.
2. Open the `TowerDefenseGame` project folder.
3. Go to `src/Main.java`.
4. Click the green run button next to `public static void main`.
5. Play the game window that opens.
